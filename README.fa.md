# رزان — اپلیکیشن فول‌استک Next.js

سایت رسمی دستسازه‌های چوبی رزان — Next.js (App Router) + SQLite + پنل مدیریت روی خود دامنه.

**🌿 [مشاهده‌ی دموی زنده ←](https://razan-wood-1.onrender.com)**

## امکانات

| مسیر | توضیح |
|---|---|
| `/` | خانه: هیرو، ویترین سینمایی، منتخب محصولات، نظرات، سوالات |
| `/products/` | همه‌ی محصولات + **جستجوی زنده با ساجسشن خودکار**، فیلتر دسته، مرتب‌سازی |
| `/products/[slug]/` | صفحه‌ی اختصاصی هر محصول: گالری، لایت‌باکس، کارهای مشابه |
| `/gallery/` | گالری آجری همه‌ی عکس‌ها |
| `/custom-order/` | فرم سفارش طرح دلخواه → پیام آماده برای واتساپ/دایرکت |
| `/about/` | قصه‌ی برند و مراحل ساخت |
| **`/admin`** | **پنل مدیریت** با لاگین: افزودن/حذف محصول، آپلود چندعکسی (ریسایز خودکار)، مدیریت منتخب‌ها — تغییرات **بلافاصله** روی سایت می‌نشیند، بدون بیلد مجدد |

## معماری

- **Next.js 15** (App Router) — خروجی `standalone` برای هاست Node
- **SQLite** (better-sqlite3) — دیتابیس فایلی در `DATA_DIR`؛ بار اول خودکار از `lib/data/products.json` پر می‌شود
- **آپلود عکس** — با sharp ریسایز/فشرده و در `DATA_DIR/uploads` ذخیره می‌شود؛ از مسیر `/media/...` سرو می‌شود
- **احراز هویت** — رمز از env (`ADMIN_PASSWORD`)، کوکی httpOnly، محافظت با middleware
- **فونت وزیرمتن سلف‌هاست** — بدون وابستگی به گوگل

## اجرای محلی

```bash
npm install
cp .env.example .env      # و ADMIN_PASSWORD را عوض کنید
npm run dev               # http://localhost:3000  |  پنل: /admin
```

## دیپلوی روی لیارا (liara.ir)

1. در لیارا یک برنامه‌ی **Next.js** بسازید (مثلاً به اسم `razanwood`)
2. یک **دیسک** به اسم `data` بسازید و به مسیر `/app/data` مونت کنید (فایل `liara.json` همین را تعریف کرده)
3. متغیرهای محیطی را در پنل لیارا ست کنید:
   - `ADMIN_PASSWORD` = یک رمز قوی
   - `DATA_DIR` = `/app/data`
4. دیپلوی:
   ```bash
   npm i -g @liara/cli
   liara deploy
   ```
5. در بخش دامنه‌های لیارا، `razanwood.ir` را اضافه کنید و رکورد DNS را طبق راهنمایش در ایرنیک/پنل دامنه ست کنید — SSL رایگان خودکار فعال می‌شود

روی هر هاست Node دیگر (پارس‌پک کلود، VPS و…) هم با `npm run build && npm start` بالا می‌آید — فقط `ADMIN_PASSWORD` و `DATA_DIR` (یک پوشه‌ی دائمی) را ست کنید.

## دیتا

- محصولات: دیتابیس SQLite در `DATA_DIR/razan.db` (از پنل `/admin` مدیریت کنید)
- seed اولیه: `lib/data/products.json`
- نظرات و سوالات: `lib/data/reviews.json` و `lib/data/faqs.json`
- لینک‌ها و شماره واتساپ: `lib/site.js`
